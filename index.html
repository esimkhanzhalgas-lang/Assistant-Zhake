Менің тәжірибем

Жұмысты орындауда қандай құрал көмектесті? Бұл жобаны жасауда маған Groq (Llama 3.1) және ChatGPT құралдары көмектесті. Олар HTML құрылымын жылдам құруға және CSS анимацияларын реттеуге мүмкіндік берді.

ЖИ нені дұрыс жасамады? Кейде ЖИ API мекенжайларын ескі нұсқада беріп қойды, мен оны Groq құжаттамасына қарап, fetch сұраныстарын қолмен түзеттім.

5 жылдан кейін не болады? ЖИ 5 жылдан кейін әзірлеушілерді алмастырмайды, бірақ "код жазушы" мамандығын "жүйе архитекторы" деңгейіне көтереді. Программистер код жазбай, ЖИ-ге тапсырма берумен айналысады.
