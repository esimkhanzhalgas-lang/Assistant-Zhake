# Assistant-Zhake
Менің бірінші ЖИ-ассистентім
Бұл — веб-әзірлеуде жасанды интеллект құралдарын қалай қолдануға болатынын көрсететін практикалық жоба. Жоба аясында мен ЖИ-құралдарымен танысып қана қоймай, олардың "капот астында" қалай жұмыс істейтінін түсінуге тырыстым.  
Жоба туралы
Бетте әзірлеушілерге арналған негізгі ЖИ-құралдары (GitHub Copilot, ChatGPT, Claude) туралы ақпарат берілген. Сондай-ақ, пайдаланушы ЖИ-мен тікелей сөйлесе алатын чат-интерфейсі орнатылған.  
Пайдаланылған құралдар
* HTML5: Беттің құрылымын жасау үшін.  
* CSS3: Адаптивті дизайн (Flexbox/Grid) және интерактивті эффектілер (hover) үшін.  
* JavaScript: ЖИ API-мен байланыс орнату, қараңғы тақырыпты қосу және сұраныстарды санау үшін.  
Қалай іске қосу керек
1.	Репозиторийді компьютерге жүктеп алыңыз немесе клондаңыз.
2.	Браузер арқылы index.html файлын ашыңыз.  
3.	Ешқандай қосымша фреймворк орнатудың қажеті жоқ (таза HTML/CSS/JS).  
Мен не үйрендім
* ЖИ құралдарын нақты веб-жобаға интеграциялауды үйрендім.  
* CSS арқылы мобильді және десктоптық нұсқаларға арналған адаптивті бет жасауды меңгердім.  
* API арқылы деректер алмасудың негізгі принциптерін түсіндім.  
Менің тәжірибем (Рефлексия)
* Ең көп көмектескен құрал: Жұмыс барысында ChatGPT кодтың логикасын түсіндіруде көп көмектесті.  
* ЖИ қателігі: Кейде ЖИ ескірген функцияларды ұсынды, мен оларды MDN құжаттамасы арқылы түзеттім.  
* Болашаққа болжам: 5 жылдан кейін ЖИ әзірлеушілердің орнын баспайды, бірақ жұмысты бірнеше есе жылдамдататын таптырмас көмекшіге айналады деп ойлаймын.
