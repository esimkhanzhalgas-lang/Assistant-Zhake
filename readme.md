# ЖИ Құралдары - Веб-бет

## 📖 Жоба туралы

Бұл жоба — жасанды интеллект құралдарын таныстыратын және Claude API арқылы нақты уақыт режимінде чат-ассистентпен сөйлесуге мүмкіндік беретін интерактивті веб-бет. Жоба заманауи веб-технологияларды (HTML5, CSS3, JavaScript ES6+) қолдана отырып жасалған.

### Негізгі мүмкіндіктер:
- 🎯 **ЖИ құралдарының карточкалары** — GitHub Copilot, ChatGPT, Claude туралы ақпарат
- 🌓 **Қараңғы/жарық тақырып** — CSS айнымалылары арқылы тақырыпты ауыстыру
- 💬 **ЖИ Чат** — Claude API интеграциясы, кез келген сұраққа жауап беру
- 📊 **Сұраныс санауышы** — жіберілген сұраныстарды есептеу
- ✨ **Анимациялар** — карточкалардың пайда болуы, хабарламалар, жүктеу индикаторы
- 📱 **Адаптивті дизайн** — мобильді және десктоп құрылғыларға бейімделген
