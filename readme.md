# ЖИ веб-әзірлеуде: Практикалық жұмыс №1

### Жоба туралы
Бұл жоба әзірлеушілерге арналған ЖИ құралдарын таныстыруға және API арқылы чат-ботты біріктіруге арналған.

### Қалай іске қосу керек
1. Файлдарды (index.html, style.css, script.js) бір папкаға жүктеңіз.
2. `index.html` файлын браузер арқылы ашыңыз.

### Пайдаланылған құралдар
- HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6).
- Groq Cloud API (Llama 3.1 моделі).
- FontAwesome (иконкалар үшін).

### Мен не үйрендім
- API-мен жұмыс істеу (fetch сұраныстары).
- CSS-тегі адаптивті дизайн және қараңғы тақырыпты енгізу.
- Intersection Observer арқылы анимация жасау.
